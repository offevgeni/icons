📥 Как подключить к проекту

Выполните эту команду в терминале вашего проекта, чтобы установить библиотеку иконок:
Установка:
npm install @offevgeni/icons@latest скопировать
Теперь вы можете использовать иконки в Svelte-компонентах:
Импорт:
import { Heart, Icon } from '@offevgeni/icons'; скопировать
По имени:
<Icon name="heart" size={24} color="red" /> скопировать
Напрямую:
<Heart size={24} color="currentColor" /> скопировать

🔄 Как обновлять иконки в проекте

После публикации новой версии выполните в проекте-потребителе:
Обновить:
npm update @offevgeni/icons скопировать
💡 Совет: чтобы всегда получать последнюю версию автоматически, в package.json вашего проекта укажите:
"@offevgeni/icons": "^1.2.4" скопировать

📤 Опубликовать новую версию

⚙️ Предварительная настройка (один раз)

Чтобы публикация работала, нужно один раз настроить авторизацию в npm:
  1. Зарегистрируйтесь на npmjs.com (если ещё нет аккаунта)
  2. В терминале этого проекта выполните: npm login
  3. Введите логин, пароль и email
  4. Готово! Теперь кнопка «Опубликовать» будет работать
Для GitHub Packages (альтернатива):
  1. Создайте Personal Access Token на github.com/settings/tokens с правами write:packages
  2. Выполните: npm login --registry=https://npm.pkg.github.com
  3. Username — ваш GitHub-логин, Password — токен
