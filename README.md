# @offevgeni/icons

Библиотека SVG-иконок для Svelte 5.

---

## 🖼 Как добавить иконки (через браузер)

1. Откройте репозиторий на GitHub
2. Перейдите в папку `svg/`
3. Нажмите **Add file → Upload files**
4. Перетащите `.svg` файлы и нажмите **Commit changes**
5. Готово! GitHub Actions автоматически:
   - соберёт компоненты
   - поднимет версию (patch)
   - опубликует в GitHub Packages

Статус сборки: **Actions → Build & Publish Icons**
